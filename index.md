# 📚 导语

我曾经写下一些字，零散地分布在泛黄的纸张上，淹没在回南天的水汽酿成的霉雾里，最后消失在回收废品的货郎高高翘起的秤砣另一头。

后来，字们找到一个空间暂存，却随着QQ的没落逐渐被我遗忘在脑后。

从2020年开始，很多个哄睡完孩子的夜晚，我在微弱的键盘光的陪伴下，用断断续续的敲击声把文字镌刻在简书上。

生下老三前，我与简书暂别，去年回归，总觉着少了点什么感觉，于是短暂重逢又离开。

马年来临之际，对输出文字的渴望又一次涌上心头。这一次，我将建立一个独属于自己的海岛——永冬泩。

我们，一群简书上神交的文字好友，曾经畅想用文字构建一个岛屿，一个群落，并命名为永冬泩。我曾畅想过拥有一个这样真实的岛屿，尽管一切从虚拟开始。告别简书之后，也许，这里是我们的下一站。

先试着从我开始，把文字一点点搬过来，拓下新足迹吧。新的一年，祝愿自己马到成功！

——2026年2月17日，马年正月初一
